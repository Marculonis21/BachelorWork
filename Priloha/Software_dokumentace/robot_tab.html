robot_tab module

GUI Robot tab

Robot tab is the second GUI tab used for selecting the robot with which we want to do the experiment (see also robots). It comes with a selection menu, image previewing the selected robot and small overview of the robot. Optionally, if the robot has any body parts which can be altered during evolution, the Select body parts for GA button will be enabled. When clicked it is going to show a menu for configuring allowed body part ranges.
