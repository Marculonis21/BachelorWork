agent_tab module

GUI Agent tab

Agent tab is the third GUI tab used for selecting and configuring genetic agent (fully described in the text of the project, see also gaAgents). User can select any agent available from the selection menu. Selected agent will display a small overview as well as its parameters in the bottom half of the tab. These parameters can be altered to change agent behaviour.

agent_tab.agents: Dict[str, BaseAgent] = {'NEATAgent': <resources.agents.gaAgents.NEATAgent object>, 'SineFuncFullAgent': <resources.agents.gaAgents.SineFuncFullAgent object>, 'SineFuncHalfAgent': <resources.agents.gaAgents.SineFuncHalfAgent object>, 'StepCycleFullAgent': <resources.agents.gaAgents.StepCycleFullAgent object>, 'StepCycleHalfAgent': <resources.agents.gaAgents.StepCycleHalfAgent object>, 'TFSAgent': <resources.agents.gaAgents.TFSAgent object>}

Dictionary of all available agents by name.

Warning

User must include new custom agents inside this dictionary to make them available through GUI. The source code gives an example how to add new entries.

agent_tab.agents_argument_options = {'NEATAgent': <PySimpleGUI.PySimpleGUI.Column object>, 'SineFuncFullAgent': <PySimpleGUI.PySimpleGUI.Column object>, 'SineFuncHalfAgent': <PySimpleGUI.PySimpleGUI.Column object>, 'StepCycleFullAgent': <PySimpleGUI.PySimpleGUI.Column object>, 'StepCycleHalfAgent': <PySimpleGUI.PySimpleGUI.Column object>, 'TFSAgent': <PySimpleGUI.PySimpleGUI.Column object>}

Dictionary of agent arguments corresponding to each agent. Used when selecting different agent from the selection menu to show the correct arguments for current agent and to hide all other ones.

This dictionary is filled automatically from the :attr: dictionary.
