experiment_params module

class experiment_params.ExperimentParams(robot: BaseRobot, agent: BaseAgent, population_size=100, generation_count=150, show_best=False, save_best=True, save_dir='./saves/experiment_runs/', show_graph=True, note='')

Class for holding all experiment parameters that are used when setting up and running evolutionary algorithms.

Parameters:

  • robot (BaseRobot) – Selected robot instance.

  • agent (BaseAgent) – Selected agent instance.

  • population_size (int) – Size of a population for evolutionary algorithm.

  • generation_count (int) – Number of generations for evolutionary algorithm.

  • show_best (bool) – Flag determining if best solution should be rendered after the experiment is done.

  • save_best (bool) – Flag determining if best individual should be saved (now always True).

  • save_dir (str) – Path of save dictionary (often auto-generated)

  • show_graph (bool) – Flag determining if graph should be shown during experiment run.

  • note (str) – Optional parameter used when saving experiment to customize folder name.
