experiment_setter module

Experiment setter module

Module used for setting up, saving and loading experiments. Experiments are stored in dictionary as (name, ExperimentParams) pairs.

See also

experiment_params : Module defining class of experiment parameters.

Usage example:

from experiment_setter import Experiments
experiments = Experiments()
...
params = experiments.get_experiment("...")
class experiment_setter.Experiments

Class of user defined experiments. Experiment parameters for main roboEvo.run_experiment() method are accessed via class methods or via get_experiment() method which takes an experiment name as an argument.

When defining new experiments users need to create their own method for parameter creation and then add the experiment to __experiments dictionary with selected custom name.
