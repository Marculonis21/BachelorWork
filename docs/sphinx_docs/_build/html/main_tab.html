main_tab module

GUI Main tab

This is the first of four tabs that opens with the graphical user interface. On this tab the user can see small overview of current experiment configuration. Then there are buttons for saving current configuration and loading previously created ones. User can also open a popup for viewing individual runs. Lastly, user can set save directory for the experiment results and start the current experiment.
